Define all maximum of all sliding windows size k

Data structure - Dequeue

Posted on 2020, Nov 18 by Tracey Hoang 3 mins read
Table of Contents
    • Problem description
    • Algorithm

Source code

Problem description

Given an array of n elements. Define the maximum value of every sliding window size k of that array. A window size k of the array is any subarray of k elements of original array.

Example:
Input: arr = {7, 6, 5, 4, 8, 9, 1}, and window size k = 3
Output: {7, 6, 8, 9, 9}

Algorithm

1. Naive algorithm

Naive algorithm is to scan each window separately, and find the maximum value of each window.

Time complexity: \( O(n \times k) \)
Space complexity: \( O(1) \)

Implementation

func maxOfWindow(arr []int32, m int32) []int32 {
	n := int32(len(arr))
	res := make([]int32, 0, n)
	for i := int32(0); i <= n - m; i++ {
		max := arr[i]
		for j := i+1; j < i+m; j++ {
			if max < arr[j] {
				max = arr[j]
			}
		}
		res = append(res, max)
	}
	return res
}

2. Dequeue algorithm

We’re going to use dequeue to solve this problem in \( O(n) \) in term of time complexity, and \( O(k) \) in term of space complexity. Dequeue is a double-ended queue, that allows removing element at its both ends.

The idea is storing only relevant elements in a dequeue. Before adding a new element, we will drop all smaller elements in that dequeue. The dequeue, however, doesn’t store elements’ values but their indices in original array. The reason of storing the indices is to easily check whether or not an element belong to the current window. More detail about algorithm and its complexity can be found at www.geeksforgeeks.org and https://stackoverflow.com)

Let’s go through an example to visualise the algorithm. Given an array arr = {7, 6, 5, 4, 8, 9, 1}, and k = 3. In the illustration below, the current window size of 3 is display in orange, dequeue in blue, and result in green. It is noted that the dequeue is holding the actual values of elements instead of their indices so that it’s easier to follow.

Implementation

  • Create a struct deque and its methods push(), popEnd(), popStart()
  type deque struct {
	index []int32
}

func (l *deque) popEnd() {
	n := len((*l).index)
	(*l).index = (*l).index[:n-1]
}

func (l *deque) popStart() {
	(*l).index = (*l).index[1:]
}

func (l *deque) push(a int32) {
	(*l).index = append((*l).index, a)
}
  • Create a dequeue variable. Its first element is 0 as the first relevant element is at index i = 0
dequeue := new(deque)
dequeue.push(0)
  • For the first window (the first k elements): run a loop from 1 to k-1 and insert relevant elements from original array arr to dequeue. At iteration i, all the indices whose corresponding values are smaller than arr[i] are removed from the dequeue, and then i is pushed to the dequeue.
for i := int32(1); i < k; i++ {
		for len(dequeue.index) != 0 && arr[i] >= arr[dequeue.index[len(dequeue.index)-1]] {
			dequeue.popEnd()
		}
		dequeue.push(i)
	}
  • After finishing iteration through the first window, the most-left element in dequeue is the index of maximum element of that window, we then include that value to our array res that will be returned at the end of program.
res := make([]int32, 0, n)
res = append(res, arr[dequeue.index[0]])
  • Move to next window by running the loop from k to the end. First thing is to remove the front element of dequeue if it is out of the current window. Repeat all the steps that we have done for the first window.
	for i := k; i < n; i++ {
		if dequeue.index[0] < i  -k + 1 {
			dequeue.popStart()
		}
		for len(dequeue.index) != 0 && arr[i] >= arr[dequeue.index[len(dequeue.index)-1]] {
			dequeue.popEnd()
		}
		dequeue.push(i)
		res = append(res, arr[dequeue.index[0]])
	}
