Check for balanced brackets in a text - Stack

Data structure - Stack

Posted on 2020, Nov 17 by Tracey Hoang 3 mins read
Table of Contents
    • Algorithm
    • Implementation

Source code

Given a text that contains any brackets from the set [], {}, (). Our program is to check whether the text uses brackets correctly. If there is no mistake, print out “Balanced”. Otherwise, find index of the first unmatched closing brackets, or the last opening bracket without corresponding closing one, and print out “Unbalanced at index index”.

For example:
[()] –> Balanced
[(] –> Unbalanced at index 3
[() –> Unbalanced at index 1

Algorithm

We are going to use Stack in this problem. Stack is a type of data structure that follows the Last-in-first-out (LIFO) principle. Based on LIFO principle, insertion to and deletion from a stack are made at its end. An example of stack is that you place some dining plates in a stack from bottom to top, and when you want to use a plate, you will take out the top one.

We then traverse through the text, everytime we meet an opening bracket, we push it into stack. On the other hand, everytime we meet a closing bracket, we check whether its corresponding opening brackets is matched with the one at the end of the stack. If there is such a match, we will pop() the last element out of the stack, and continue traversing. Otherwise, print out Unbalanced with index of the closing bracket.

In case there is a missing closing brackets, we will print out the index of the last opening brackets without corresponding closing one.

Implementation

Define a stack type with value field is an array of string of the opening brackets, and index field is an array of their indices.

type stack struct {
	value []string
	index []int
}

The stack has 3 methods, push(bracket string, index int) is used to push bracket and its index into stack; pop() is to pop the last element out of stack; and isEmpty() is to check whether stack is empty.

// isEmpty: check if stack is empty
func (s *stack) isEmpty() bool {
	return len((*s).value) == 0
}

// Push a new key onto the stack
func (s *stack) push(bracket string, index int) {
	(*s).value = append((*s).value, bracket)
	(*s).index = append((*s).index, index)
}

// Remove top element of stack
func (s *stack)	pop() string {
	if s.isEmpty() {
		return "false"
	}
	i := len((*s).value)-1
	res := (*s).value[i]
	(*s).value = (*s).value[:i]
	(*s).index = (*s).index[:i]
	return res
}

Now, we will traverse the text, and find out is there any unmatched brackets.

func checkBrackets(s []string) string {
	var brackets stack
	for i , v := range s {
		if v == "[" || v == "{" || v == "(" {
			brackets.push(v,i+1)
		} else if v == ")" || v == "]" || v == "}" {
            res := brackets.pop()
            // Check is there any corresponding opening bracket of v, 
            // or whether the closing bracket is matched with the opening one.
            // If "false" or unmatched, return "Unbalanced"
			if res == "false" || res != matchS(v) {
				return "Unbalanced at index "+strconv.Itoa(i+1)
			}
		}
    }
    // Check to see is there any bracket without corresponding closing one
	if !brackets.isEmpty() {
		return "Unbalanced at index "+strconv.Itoa(brackets.index[len(brackets.index)-1])
	}
	return "Balanced"
}

// Utility function
func matchS(s string) string {
	if s == ")" {
		return "("
	}
	if s == "]" {
		return "["
	}
	return "{"
}
