Iterative method to traverse a tree

Traverse the binary tree (or graph) - iterative method

Posted on 2021, Feb 20 by Tracey Hoang 5 mins read
Table of Contents
    • Problem description
    • Implementation

Source code

Problem description

In previous post we have already traversed the tree using recursive method (see link). Recursive method is neat in term of length of code, and easy to understand. Recursion, however, runs slower than iteration, especially when the tree is very deep that requires a considerable number of recursive calls.

In this post, we will traverse a binary tree using iterative method, in which all tree nodes are iterated over in some manner. There are more than one possible next nodes to visit, so they need to be deferred. For that purpose, an extra data structure, such as stack or queue, is required to store the deferred nodes.

Implementation

We will consider both depth-first traversal and breadth-first traversal.

Depth-first traversal (in-order, pre-order, and post-order traversal)

For depth-first traversal of the tree, we are going to use stack

type stack []node

func (s *stack) push(n node) {
	*s = append(*s, n)
}

func (s *stack) pop() *node {
	a := (*s)[len(*s)-1]
	*s = (*s)[:len(*s)-1]
	return &a
}

func (s *stack) peek() node {
	return (*s)[len(*s)-1]
}

func (s *stack) isEmpty() bool {
	return len(*s) == 0
}
In-order traversal
  1. Create an empty stack s, and initialise current node as root;
  2. While curr is not nil or stack is not empty, then:
    • push the current node to stack s, and set curr = curr.left until there is no left child;
    • pop the item from s and set curr = s.pop, either put item’s key to our answer array (if you want return an array) nor just print out the item’s key;
    • set curr = curr.right
func inOrderTraversal(arr []node) []string {
	s := &stack{}
	curr := &arr[0]
	answer := make([]string,0,len(arr))
	for curr != nil || !s.isEmpty() {
		for curr != nil {
			s.push(*curr)
			curr = curr.left
		}
		curr = s.pop()
		answer = append(answer,curr.key)
		curr = curr.right
	}
	return answer
}
Pre-order traversal
  1. Create an empty stack s and push the root node to the stack;
  2. While the stack s is not empty:
    • pop the item from stack s, and put it (or print it out) to the answer array;
    • push the right child of the popped item to stack;
    • push the left child of the popped item to stack.
func preOrderTraversal(arr []node) []string {
	s := &stack{}
	s.push(arr[0])
	answer := make([]string,0, len(arr))
	for !s.isEmpty() {
		c := s.pop()
		answer = append(answer,c.key)
		if c.right != nil {
			s.push(*c.right)
		}
		if c.left != nil {
			s.push(*c.left)
		}
	}
	return answer
}
Post-order traversal

For post-order traversal, we use 2 stacks: first stack s to store the deferred node as the one in the pre-order traversal, and a second stack s1 to store the reverse post-order traversal.

  1. Create 2 stacks s and s1;
  2. Push root node into the first stack s;
  3. While s is not empty:
    • pop an item from first stack s;
    • instead of print out the popped item (or put it into the answer array), we push it into the second stack s1;
    • push the left child of the popped item into first stack s;
    • push the right child of the popped item into first stack s;
  4. While s1 is not empty, we keep popping the item from it and put the item to the answer array or print it out.
func postOrderTraversal(arr []node) []string {
	s := &stack{}
	s1 := &stack{}
	s.push(arr[0])

	for !s.isEmpty() {
		c := s.pop()
		s1.push(c)
		
		if c.left != nil {
			s.push(*c.left)
		}
		if c.right != nil {
			s.push(*c.right)
		}
	}
	answer := make([]string, 0, len(arr))
	for !s1.isEmpty() {
		c := s1.pop()
		answer = append(answer, c.key)
	}
	return answer
}

Breadth-first traversal

For breadth-first traversal of the tree we are going to use queue.

type queue []node
func newQueue() *queue{
	return &queue{}
}
func (q *queue) isEmpty() bool {
	return len(*q) == 0
}
func (q *queue) push(v node) {
	*q = append(*q,v)
}
func (q * queue) pop() node {
	res := (*q)[0]
	*q = (*q)[1:]
	return res
}
  1. Create a queue q, then push the root tree into the queue, and put its value into the answer array;
  2. While the queue is not empty:
    • pop the item from q;
    • if left child is not nil, push the left child to the queue and its value to the answer array;
    • if right child is not nil, push the right child to the queue and its value to the answer array.
func bfs(arr []node) []string {
	q := newQueue()
	q.push(arr[0])
	answer := make([]string, 0, len(arr))
	answer = append(answer, arr[0].key)
	for !q.isEmpty() {
		c := q.pop()
		if c.left != nil {
			q.push(*c.left)
			answer = append(answer,c.left.key)
		}
		if c.right != nil {
			q.push(*c.right)
			answer = append(answer,c.right.key)
		}
	}
	return answer
}

Our program will read the input from STDIN and then print out the output. We are given a binary tree. The nodes of the tree are numbered from 0 to n-1. Node 0 is the root. The first line contains the number of nodes n. The next n lines contain information about nodes. In each line, the first value is the node’s key, then follows by 2 other values: keys of the node’s left child and right child. If the node doesn’t have left or right child, the corresponding key will be -1.

For example, the following input can be illustrated with corresponding tree:

5
4 1 2
2 3 4
5 -1 -1
1 -1 -1
3 -1 -1

Output printed out when traverse the tree:

  • in-order traversal: {1, 2, 3, 4, 5}
  • pre-order traversal: {4, 2, 1, 3, 5}
  • post-order traversal: {1, 3, 2, 5, 4}
  • breadth-first traversal: {4, 2, 5, 1, 3}
func main() {
	scanner := bufio.NewScanner(os.Stdin)
	const maxCapacity = 2048 * 2048
	buf := make([]byte, maxCapacity)
	scanner.Buffer(buf, maxCapacity)
	scanner.Scan()
	// number of nodes in the tree
	n, _ := strconv.ParseInt(scanner.Text(), 10, 64)
	arr := make([]node, n)
	// make tree
	for i := int64(0); i < n; i++ {
		scanner.Scan()
		data := strings.Split(scanner.Text(), " ")
		arr[i].key = data[0]
		j, _ := strconv.ParseInt(data[1], 10, 64)
		k, _ := strconv.ParseInt(data[2], 10, 64)
		if j == -1 {
			arr[i].left = nil
		} else {
			arr[i].left = &arr[j]
		}
		if k == -1 {
			arr[i].right = nil
		} else {
			arr[i].right = &arr[k]
		}
	}
	// inOrder
	fmt.Println("In-Order traversal")
	fmt.Println(inOrderTraversal(arr))

	// preOrder
	fmt.Println("pre-Order traversal")
	fmt.Println(preOrderTraversal(arr))

	// postOrder
	fmt.Println("post-Order traversal")
	fmt.Println(postOrderTraversal(arr))

	// breadth-first search
	fmt.Println("breadth-first traversal")
	fmt.Println(bfs(arr))
}
