Majority Element Of An Array

Implemented by different approaches

Posted on 2020, Oct 24 by Tracey Hoang 3 mins read
Table of Contents
    • Problem description
    • Implementation

Source code

Problem description

An array is said to have a majority element if more than half of its elements are the same.

Given a non-empty array arr. Check to see if the array has a majority element.

Implementation

Let’s have a look at some approaches of solving the given problem.

1- Brute Force algorithm

This approach is to find the occurrences of each element. As soon as there is an occurrence of an element is greater than a half, we know that this array has majority element and then return true. Otherwise, return false

func bruteForce(arr []int32) bool {
	n := len(arr)
	for _, v:= range arr {
		count := 0
		for i := 0; i < n; i++ {
			if arr[i] == v {
				count += 1
			}
		}
		if count > n/2 {
			return true
		}
	}
	return false
}

Time complextiy \( O(n^2) \).
Space complexity \( O(1) \).

2- Sorting

If the array is sorted, and if that array has majority element, that element should be located at index n/2 where n is the total number of elements in the array (length of the array). So all we need to do is checking whether all the elements are the same and equal to element at index n/2 in the sliding window’s size at least n/2+1. If yes, return true, otherwise, return false.

func sortArray(arr []int32) bool {
	sort.Slice(arr, func(i, j int) bool {
		return arr[i] < arr[j]
	})
	n := len(arr)
	for i := 0; i < n/2; i++ {
		if arr[i] == arr[n/2] {
			if arr[i+n/2] == arr[n/2] {
				return true
			}
		}
	}
	return false
}

As we can see, the time and space complexities of above approach are the time and space complexities of sorting algorithm that we used. For example, in case you sort the array by applying merge sort algorithm, the time complexity will be \( O(n\log n) \), and space complexity will be \( O(n) \).

3- Using Map

You can reduce time complexity to \( O(n) \) while maintaining space complexity \( O(n) \) by using a map. It is quite straightforward approach. We create a map with keys are array elements, and its values are element’s occurrences. As soon as any value of a key is greater than n/2, the program returns.

func useMap(arr []int32) bool {
	n := int32(len(arr))
	m := make(map[int32]int32)
	for _,v := range arr {
		m[v] += 1
		if m[v] > n/2 {
			return true
		}
	}
	return false
}

5- Boyer-Moore algorithm
func BoyerMoore(arr []int32) bool {
	count := 1
	currentNumber := arr[0]
	for i := 1; i < len(arr); i++ {
		if count == 0 {
			currentNumber = arr[i]
		}
		if arr[i] == currentNumber {
			count += 1
		} else {
			count -= 1
		}
	}

	count = 0
	for _, v := range arr {
		if v == currentNumber {
			count += 1
		}
	}
	if count > len(arr)/2 {
		return true
	}
	return false
}

Time complextiy \( O(n) \).
Space complexity \( O(1) \).
