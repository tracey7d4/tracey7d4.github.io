This post is about some problems that can be solved easily only by using math formula.

1- Greatest common divisor and Least Common Multiple

The greatest common divisor \( GCD(a,b) \) of two non-negative integers \( a \) and \( b \) (which are not both equal to 0) is the greatest integer that divides both \( a \) and \( b \).

The least common multiple of two positive integers \( a \) and \( b \) is the least positive integer that is divisible by both \( a \) and \( b \).

You can compute \( GCD(a,b) \) by implementing Euclidean algorithm, which is defind as follows \[ GCD(a,0) = a \] \[ GCD(a,b) = GCD(b,r) \] where \( a = b\times q + r \)

On the other hand, there is a close relation between \( GCD(a,b) \) and \( LCM(a,b) \) \[ LCM(a,b) = {a\times b \over GCD(a,b)} \]

Implementation

Greatest common divisor

func GCD(a,b int64) int64 {
	r := int64(1)
	for r !=0 {
		r = a%b
		if r == 0 {
			return b
		}
		a,b = b,r
	}
	return a/b
}

Least common multiple

func LCM(a,b int64) int64 {
	return a*b/GCD(a,b)
}

2- Fibonacci number and related problems

Fibonacci numbers are defined as: \( F_0 = 0, F_1 = 1 \), and \( F_i = F_{i-1} + F_{i-2} \) for \( \forall i \ge 2 \).

func fibonacci(n int) int64 {
	if n <= 1 {
		return n
	}
	a := int64(0)
	b := int64(1)
	var fi int64
	for i := 2; i <= n; i++ {
		fi = a + b
		a,b = b, fi
	}
	return fi
}

As Fibonacci number grows very fast and soon overload with type int64. For convenience, we only computes the last digit of each problems so on. Hence, the above fibonacci() is modified as follows.

func lastDigitFibonacci(n int) int {
	if n == 0 {
		return 0
	}
	if n == 1 {
		return 1
	}
	a := 0
	b := 1
	var fi int
	for i := 2; i <= n; i++ {
		fi = (a + b)%10
		a,b = b, fi
	}
	return fi
}

Now let’s have a look at some other Fibonacci problems.

Sum of the first n Fibonacci numbers \( F_0 + F_1 + ... + F_n\)

\[ \sum_{i=0}^{i=n} {F_i} = F_{n+2}-1\] 
func lastDigitSum(n int) int {
	return lastDigitFibonacci(n+2)-1
}

Partial sum of Fibonacci numbers \( F_m + F_{m+1} + ... + F_n\) \[ \sum_{i=m}^{i=n} {F_i} = F_{n+2}-F_{m+1} \]

func lastDigitPartialSum(m,n int64) int {
	a := lastDigitFibonacci(n+2)
	b := lastDigitFibonacci(m+1)
	if a > b {
		return a-b
	}
	return a+10-b
}

Sum of square of Fibonacci numbers \( F_0^2 + F_1^2 + ... + F_n^2\)

\[ \sum_{i=0}^{i=n} {F_i^2} = F_{n} \times F_{n+1}\] 
func lastDigitSquareSumFib(n int64) int {
	if n < 2 {
		return int(n)
	}
	a := 0
	b := 1
	for i := int64(2); i <= n+1; i++ {
		a, b = b,(a + b)%10
	}
	return (a*b)%10
}
